快速安装与部署

秒级接入你的开发环境。选择目标平台开始：

powershell 
# 请在 PowerShell 中以管理员身份运行
iwr -useb https://raw.githubusercontent.com/xjwm5685-ui/cliai/main/install.ps1 | iex
bash 
# 一键安装、配置 zsh 环境并启动
curl -fsSL https://raw.githubusercontent.com/xjwm5685-ui/cliai/main/install.sh | bash && sudo apt update && sudo apt install -y cliai zsh && cliai shell install zsh && exec zsh
bash (Intel) 
# Intel Mac 一键下载安装并配置 zsh 环境
mkdir -p ~/cliai-install && cd ~/cliai-install
curl -LO https://github.com/xjwm5685-ui/cliai/releases/latest/download/cliai_macOS_x86_64.tar.gz
tar -xzf cliai_macOS_x86_64.tar.gz
./scripts/install-unix.sh
cliai shell install zsh
exec zsh
bash (Apple Silicon) 
# Apple Silicon Mac (M1/M2/M3) 一键下载安装并配置 zsh 环境
mkdir -p ~/cliai-install && cd ~/cliai-install
curl -LO https://github.com/xjwm5685-ui/cliai/releases/latest/download/cliai_macOS_ARM64.tar.gz
tar -xzf cliai_macOS_ARM64.tar.gz
./scripts/install-unix.sh
cliai shell install zsh
exec zsh

或者直接前往 GitHub Releases 下载预编译二进制文件。
